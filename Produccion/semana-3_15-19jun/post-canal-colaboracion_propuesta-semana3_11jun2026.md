**📅 Pub -- Técnico · Canal Primeros Pasos (semana y tema por confirmar)**

**Cómo hacer que GitHub Copilot conozca tu entorno de equipo**

GitHub Copilot no nace sabiendo las reglas de tu proyecto. Pero se puede configurar para que las conozca: qué lenguajes usan, qué restricciones tienen, qué herramientas del equipo puede consultar. Eso se hace con Skills y con un archivo de instrucciones del equipo.

El resultado: respuestas más precisas desde el primer prompt, sin tener que explicar el contexto cada vez.



Preguntas acerca de este tema:

 ¿Qué Skills se priorizarian para un developer de GCO en Visual Studio? ¿Jira, Confluence, alguna otra?
 ¿Hay alguna herramienta que estén recomendando o usando ya en las sesiones de formación?, o ¿Qué proponen a respecto?
