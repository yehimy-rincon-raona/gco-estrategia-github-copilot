# _propuestas/ — Backlog de contenido pendiente de asignación

Aquí van las ideas de contenido que ya tienen concepto y brief pero todavía no tienen semana ni pub asignada.

**Flujo:**
1. Alguien propone un tema → se crea el brief propuesta aquí
2. Nibaldo valida el enfoque técnico → deja comentario o aprueba
3. Se asigna semana y pub en el TABLERO → el archivo se mueve a `semana-N_fechas/pub-0X_/`

**Organización por eje temático:**

| Carpeta | Eje |
|---------|-----|
| `eje_novedades-y-tips/` | Novedades, Actualizaciones y Tips de uso |
| `eje_casos-de-uso/` | Casos de uso reales en GCO |
| `eje_primeros-pasos/` | Configuración y primeros pasos |
