## Nota de producción — leer antes de generar la voz

Este documento cubre únicamente el guion de voz en off para la sección 5 de la grabación de pantalla existente (archivos de Nibaldo en `_grabaciones/`). El editor sincroniza la voz con el video de pantalla en postproducción.

**Cómo generar la voz:**
La voz en off se genera con **Clipchamp**, usando la voz **Alessio Dragon HD Latest**. Copiar únicamente el texto del bloque de voz en off (sin comillas ni encabezados) y pegarlo en el campo de texto de Clipchamp. Si alguna frase queda con ritmo extraño, dividirla en dos bloques separados.

**Datos sensibles:**
En el video de pantalla pueden aparecer datos del usuario de Nibaldo. Difuminar en edición: nombre de usuario de GitHub y código de dispositivo si se considera sensible.

### Sección 5 · Autenticación desde la terminal

**Voz en off:**
> "Para usar GitHub Copilot desde la terminal, abre una terminal dentro del proyecto específico con el que vas a trabajar — no en la carpeta general, sino dentro del proyecto.
>
> Escribe barra diagonal para ver los comandos disponibles, selecciona login y GitHub Copilot te preguntará cómo quieres autenticarte. Elige github.com.
>
> Te va a mostrar un código de dispositivo. Abre el enlace con Control clic, pega el código y autoriza el acceso. Cuando la terminal confirme que la sesión está iniciada, GitHub Copilot ya está listo para responder desde consola.
>
> Eso es todo. Entorno configurado."

**Qué mostrar en pantalla:**
Fragmento de la grabación de Nibaldo correspondiente a la sección 5. Debe verse la terminal abierta dentro de un proyecto específico — no en la carpeta padre. A continuación el flujo completo: barra diagonal para ver los comandos, selección de login, elección de github.com, código de dispositivo en pantalla, apertura del enlace en el navegador, pegado del código y mensaje de confirmación de sesión iniciada en la terminal.
