## ¿Para qué sirve esta comprobación?

Evita bloqueos iniciales y permite empezar a trabajar con GitHub Copilot de forma controlada.

---

> **Antes de empezar — dos cosas importantes:**
>
> Visual Studio ya está instalado en tu ordenador corporativo. No necesitas descargarlo ni instalarlo — el icono está en el escritorio.
>
> Si GitHub Copilot no aparece activo al abrir Visual Studio, puede que tu licencia no esté activada todavía. Ve a **My Profile de Microsoft** → menú lateral → 2 clics → "Solicitar acceso". En ~40 minutos estará disponible en tu IDE.

---

## 1. Abre Visual Studio

Doble clic en el icono del escritorio. Si no lo encuentras, búscalo en el menú de inicio.

No necesitas instalar nada — Visual Studio ya está en tu máquina.

---

## 2. ¿Necesitas instalar algo más?

Si necesitas Git, una actualización de Visual Studio o el propio GitHub Copilot, todo está en **Portal de Empresa** — el aplicativo corporativo disponible en tu ordenador o escritorio remoto.

Abre Portal de Empresa → busca lo que necesitas → instala desde ahí.

---

## 3. Valida GitHub Copilot y tu cuenta corporativa

Abre **GitHub Copilot Chat** en Visual Studio.

Si ves el campo **"Preguntar a Copilot"**, el asistente ya está activo.

Después, revisa el avatar superior derecho y confirma que estás con la cuenta corporativa de GCO.

### Comprueba que tienes:
- GitHub Copilot Chat visible.
- Cuenta corporativa correcta.

---

## 4. Abre un proyecto de prueba

Crea o abre un proyecto sencillo. No hace falta desarrollar nada todavía. Solo confirma que:

- Visual Studio carga bien.
- El proyecto se abre correctamente.
- GitHub Copilot puede trabajar con el contexto del IDE.

---

## 5. Revisa la terminal

Si vas a ejecutar comandos o scripts, confirma que usas el perfil correcto. Cuando aplique, valida **PowerShell 7** para evitar errores por diferencias entre versiones.

Comando de referencia:

```powershell
$PSVersionTable.PSVersion
```

---

## Al terminar, deberías tener:

- Visual Studio listo.
- GitHub Copilot activo.
- Cuenta corporativa validada.
- Terminal revisada.
