**Primer paso: verifica que GitHub Copilot está listo en tu Visual Studio**

Antes de escribir un prompt, comprueba que el entorno está configurado. En el **Vídeo 01 configuración del entorno** encontrarás en pocos minutos: cómo instalarlo, cómo vincular tu cuenta y el proceso de verificación.

**3 cosas que vale la pena saber desde el principio:**

- **La cuenta importa:** si el panel de GitHub Copilot aparece vacío, empieza por verificar que tienes vinculada a tu cuenta corporativa.
- **El chat está en el panel derecho:** si no lo ves, búscalo en Menú → Ver → GitHub Copilot Chat.
- **Cuanto más contexto, mejor respuesta:** desde el primer día puedes indicarle a GitHub Copilot el idioma y las convenciones de tu equipo.
