La imagen debe seguir un estilo de ilustración editorial corporativa similar al ejemplo de referencia: composición limpia, fondo blanco, pocos elementos, sensación artesanal, trazos sueltos y expresivos, mezcla de un objeto realista con dibujo lineal manual.

Requisitos visuales:
- Fondo completamente blanco.
- Debe incluir **un elemento realista** relacionado con el tema, como si fuera una fotografía recortada, objeto 3D suave o render sencillo.
- Encima o alrededor del elemento realista debe aparecer un **dibujo lineal manual**, como si estuviera hecho a mano con marcador o lápiz digital.
- El dibujo lineal debe usar siempre este color exacto: **rojo #D90028**.
- El trazo debe verse imperfecto, humano, fluido y espontáneo, no geométrico ni demasiado pulido.
- La ilustración debe ser minimalista, con mucho aire visual y sin fondo decorativo.
- No usar sombras dramáticas, degradados fuertes, texturas pesadas ni exceso de detalle.
- No incluir texto, títulos, logotipos ni marcas.
- El resultado debe parecer parte de una misma familia visual corporativa: elegante, simple, cercana y conceptual.

Dirección de estilo:
Una imagen limpia donde el objeto realista aporta contexto y el dibujo rojo aporta idea, movimiento o metáfora. El dibujo no debe cubrir todo el objeto; debe acompañarlo, rodearlo o completarlo visualmente.

Formato:
Ilustración centrada, proporción cuadrada, alta resolución, fondo blanco.
