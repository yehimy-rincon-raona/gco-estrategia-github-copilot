# Prompt — convertir icono o boceto en ilustración estilo Occident

Usa este prompt cuando quieras subir un dibujo, icono, boceto o referencia simple y convertirlo en una ilustración alineada con el estilo visual de Occident para las piezas de adopción de GitHub Copilot en GCO.

---

## Prompt base

Convierte la imagen de referencia en una ilustración vectorial limpia, institucional y cercana, siguiendo el estilo visual de Occident.

La ilustración debe mantener la idea principal del dibujo original, pero reinterpretarla con acabado profesional de marca: composición centrada, fondo blanco puro, mucho aire alrededor, líneas limpias, geometría clara y una mezcla equilibrada de rojo corporativo y grises.

Estilo visual:
- Fondo blanco, sin textura, sin degradados de fondo y sin elementos decorativos innecesarios.
- Color principal de acento: rojo Occident `#DC0028`.
- Usar grises neutros para volumen, sombras suaves y detalles técnicos: gris claro, gris medio y gris oscuro.
- El rojo debe funcionar como silueta, contorno parcial, acento o plano de apoyo visual, no como fondo sólido.
- La parte principal del objeto puede estar construida en grises, con el rojo como elemento distintivo de marca.
- Acabado tipo ilustración vectorial editorial: limpio, preciso, sin realismo fotográfico.
- Líneas finas y consistentes, con trazos técnicos y proporciones ordenadas.
- Sombreado sutil mediante planos o tramados suaves, sin sombras dramáticas.
- Sin efecto 3D exagerado, sin brillos plásticos, sin estilo cartoon, sin estética futurista.
- Bordes definidos, lectura clara incluso en tamaño pequeño.

Composición:
- Un único elemento o escena simple como protagonista.
- Centrar la ilustración dentro del lienzo.
- Dejar margen blanco amplio alrededor.
- Evitar texto dentro de la ilustración.
- Evitar iconos flotantes, fondos oscuros, fondos rojos sólidos, neones, orbes, stickers o elementos tipo clipart.

Resultado esperado:
Una ilustración sobria y moderna, coherente con las referencias de marca de Occident: objetos o conceptos representados con base gris, acento rojo `#DC0028`, fondo blanco y sensación de claridad, confianza y utilidad.

---

## Variante para piezas de GitHub Copilot / desarrollo

Convierte la imagen de referencia en una ilustración estilo Occident aplicada al contexto de desarrollo de software y adopción de GitHub Copilot.

Mantén la composición limpia y el fondo blanco. Representa el concepto de forma útil para developers: código, entorno de Visual Studio, mainframe, COBOL, DB2, PL/SQL, revisión de errores, memoria de contexto, selección de archivos o ayuda técnica, pero sin saturar la escena.

Usa rojo Occident `#DC0028` como acento de marca y grises para pantallas, herramientas, documentos, terminales, ventanas de código o elementos técnicos. La ilustración debe sentirse práctica y profesional, no promocional ni futurista.

Evita:
- Robots, cerebros digitales o clichés de inteligencia artificial.
- Fondos oscuros.
- Fondos rojos sólidos.
- Estética cyberpunk, neón o ciencia ficción.
- Exceso de elementos tecnológicos decorativos.
- Texto legible dentro de pantallas o documentos.

Busca que un developer de COBOL con 20 años en mainframe piense: "esto habla de mi trabajo, no de una moda".

---

## Prompt negativo

No usar fondo negro. No usar fondo rojo sólido. No usar estética cartoon. No usar estilo infantil. No usar brillos exagerados. No usar neones. No usar degradados morados o azules dominantes. No usar personajes tipo robot. No usar elementos de ciencia ficción. No usar texto grande dentro de la imagen. No usar composición recargada. No usar sombras duras. No usar fotografías ni mockups realistas.

---

## Checklist antes de aceptar la imagen

- ¿El fondo es blanco?
- ¿El rojo principal es `#DC0028` o visualmente equivalente?
- ¿El rojo aparece como acento/silueta y no como fondo sólido?
- ¿La ilustración combina rojo y grises?
- ¿El estilo es limpio, vectorial e institucional?
- ¿Se entiende el concepto en tamaño pequeño?
- ¿Evita clichés de IA?
- ¿Encaja con las referencias de `automotriz.png`, `hogar.png`, `medico.png` y `salud.png`?
- ¿Sirve para una pieza corta de Teams sin verse como marketing genérico?
